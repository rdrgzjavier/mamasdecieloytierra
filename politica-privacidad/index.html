Privacidad

Política de privacidad

En Mamás de Cielo y Tierra cuidamos la información que compartes con el mismo respeto con el que queremos acompañar este proyecto. Esta política explica, de forma clara, qué datos se recogen cuando solicitas información y para qué se utilizan.

Responsable del tratamiento

El responsable del tratamiento es Mamás de Cielo y Tierra. Por razones de privacidad no se publican en esta web datos personales o profesionales adicionales. Para cualquier cuestión relacionada con privacidad puedes utilizar el formulario de contacto indicando en el mensaje que se trata de una consulta sobre protección de datos.

Datos que se recogen

A través del formulario se pueden recoger nombre, email, motivo de contacto, aceptación de la política de privacidad y el mensaje que decidas escribir. El campo de mensaje es opcional: no es necesario que compartas información íntima, sanitaria o especialmente sensible para pedir información.

Finalidades

Los datos se utilizan para responder a tu solicitud, enviar información sobre el programa, gestionar una posible lista de interés o inscripción y mantener comunicaciones directamente relacionadas con Mamás de Cielo y Tierra.

Base jurídica

La base jurídica principal es tu consentimiento, que prestas al marcar la casilla de aceptación antes de enviar el formulario. Si más adelante existiera una relación de prestación de servicios, también podrían tratarse datos necesarios para gestionar esa relación.

Conservación

Los datos se conservarán durante el tiempo necesario para responder a la solicitud y gestionar la relación iniciada. Si finalmente no hay inscripción o continuidad, se eliminarán o bloquearán cuando dejen de ser necesarios, salvo obligación legal de conservación.

Destinatarios y herramientas

No se venden ni ceden tus datos a terceros con fines comerciales. La web utiliza FormSubmit para recibir los formularios por email y GitHub Pages para alojar el sitio. Estos proveedores pueden tratar datos técnicos o de formulario como encargados o proveedores necesarios para que la web funcione.

Transferencias internacionales

Algunas herramientas tecnológicas pueden operar desde fuera del Espacio Económico Europeo. En ese caso, el tratamiento se realizará conforme a las garantías previstas por la normativa aplicable.

Derechos

Puedes solicitar acceso, rectificación, supresión, oposición, limitación del tratamiento o portabilidad de tus datos. Para ejercerlos, utiliza el formulario de contacto indicando el derecho que quieres ejercer y la información necesaria para identificar tu solicitud.

Reclamaciones

Si consideras que el tratamiento de tus datos no se ajusta a la normativa, puedes presentar una reclamación ante la Agencia Española de Protección de Datos.

Cookies y analítica

La web no está pensada para instalar cookies publicitarias ni realizar perfiles comerciales. Si en el futuro se incorporan herramientas que requieran consentimiento, se actualizará esta política y se añadirá el aviso correspondiente.

Actualizaciones

Esta política puede actualizarse para reflejar cambios en el proyecto, en las herramientas utilizadas o en la normativa aplicable.
